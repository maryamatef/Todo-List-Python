# 📝 Todo List - نظام إدارة المهام

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Status](https://img.shields.io/badge/Status-Active-success.svg)]()

> تطبيق بسيط واحترافي لإدارة المهام اليومية عبر سطر الأوامر، مكتوب بلغة Python.

---

## ✨ المميزات

- ➕ **إضافة مهام جديدة** بسهولة
- 📋 **عرض جميع المهام** مع حالتها (مكتملة / قيد التنفيذ)
- ✅ **تحديد المهام المكتملة**
- 🗑️ **حذف المهام** غير المرغوبة
- 📊 **إحصائيات شاملة** عن إنتاجيتك
- 💾 **حفظ تلقائي** للبيانات في ملف JSON
- 🕒 **تسجيل وقت الإضافة والإنجاز** لكل مهمة

## 📖 شرح الكود

البرنامج يحتوي على الدوال التالية:

| الدالة | الوظيفة |
|--------|---------|
| `load_tasks()` | تحميل المهام من ملف JSON |
| `save_tasks()` | حفظ المهام في ملف JSON |
| `add_task()` | إضافة مهمة جديدة |
| `show_tasks()` | عرض جميع المهام |
| `complete_task()` | تعليم مهمة كمكتملة |
| `delete_task()` | حذف مهمة |
| `show_stats()` | عرض إحصائيات المهام |
